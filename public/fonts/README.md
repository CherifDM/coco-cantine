# Police Nagbuloe

Placez les fichiers de la police Nagbuloe dans ce dossier :

- `Nagbuloe Regular.otf` (ou `Nagbuloe Regular.ttf`)
- `Nagbuloe Bold.otf` (ou `Nagbuloe Bold.ttf`) — pour les titres et texte en gras

Le CSS charge d’abord le `.otf`, puis le `.ttf` en secours. Les noms de fichiers doivent correspondre exactement (avec l’espace).

Tant que les fichiers ne sont pas présents, le site utilise **Fredoka** (Google Fonts) comme substitut visuellement proche.
